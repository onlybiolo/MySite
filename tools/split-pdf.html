Variabili per il nome file

Scrivi il template nel campo "Nome file output" usando queste variabili. Clicca su una variabile per copiarla.

{name}
Nome del file originale (senza .pdf) Es: se il file si chiama contratto.pdf → diventa contratto
{index}
Numero progressivo del file generato Es: 01, 02, 03… (si adatta automaticamente al totale)
{range}
Pagine contenute in quel file Es: p1-5 se contiene le pagine da 1 a 5, p3 se è una pagina sola
{date}
Data di oggi Es: 2026-02-13
Esempio: con il template {name}_{range}_{date} e un file chiamato contratto.pdf ottieni:
contratto_p1-5_2026-02-13.pdf, contratto_p6-10_2026-02-13.pdf…
Variabile copiata!
